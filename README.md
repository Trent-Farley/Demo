To run this code
1. Go to your terminal
2. type 
    ```bash
    git clone https://github.com/Trent-Farley/Demo.git 
    ```
3. ```bash
    cd Demo
    ```
4. ```dotnet run```

To edit the code 

1. Open Visual Studio
2. Find the Demo folder
3. open the Demo folder
4. Click on the Refresh.csproj file
